Description

This package does combine all biopet tools into a single jar for easy access. All tools can still be used as separated jars if required, this is just to make it easier for you as user.

Installation

Executable requires Java 8 to be installed on your device. Download Java 8 here or install via your distribution's package manager.

Download the latest version of Executable here. To generate the usage run:

java -jar <Executable_jar> --help

Example

Get a list of tools:

java -jar <Executable_jar>

Normal tools:

java -jar <Executable_jar> <tool_name> <arguments to tool>

Spark tools:

spark-submit <spark arguments> <Executable_jar> <tool_name> <arguments to tool>

About

Executable is part of BIOPET tool suite that is developed at LUMC by the SASC team. Each tool in the BIOPET tool suite is meant to offer a standalone function that can be used to perform a dedicate data analysis task or added as part of BIOPET pipelines.

All tools in the BIOPET tool suite are Free/Libre and Open Source Software.

Contributing

The source code of Executable can be found here. We welcome any contributions. Bug reports, feature requests and feedback can be submitted at our issue tracker.

Executable is build using sbt. Before submitting a pull request, make sure all tests can be passed by running sbt test from the project's root. We recommend using an IDE to work on Executable. We have had good results with this IDE.
